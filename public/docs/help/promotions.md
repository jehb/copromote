# Promotions Management

Plan and track your promotional campaigns.

## Overview
This section helps you manage discounts, special offers, and seasonal campaigns.

## Creating a Promotion
Define the scope, duration, and type of promotion.
- **Code**: Unique promo code (if applicable).
- **Discount**: Percentage or fixed amount off.
- **Duration**: Start and end dates.

## Tracking
Monitor the performance of your active promotions directly from the list view.
