# Events Help

## Overview
Manage your promotional events, trade shows, conferences, and other important dates. Link events to contacts, organizations, and track logistics. You can also view external events, such as integrated market events from websites like your connected WordPress calendar, directly in your list.

## Creating an Event

1. Click **Create Event** in the top-right corner
2. Fill in event details:
   - **Title**: Event name (required)
   - **Description**: Event details and purpose
   - **Start Date/Time**: When the event begins (required)
   - **End Date/Time**: When the event ends
   - **Location**: Physical or virtual location
   - **Type**: Event category (conference, trade show, meeting, etc.)

3. Click **Create** to save the event

## Linking Contacts & Organizations

### Adding Attendees
1. Open an event
2. Click **Add Attendees**
3. Select contacts from your list
4. Contacts will be notified (if notifications are enabled)

### Linking Organizations
1. Open an event
2. Click **Add Organization**
3. Select or create an organization
4. Track which companies are involved

## Event Locations

### Creating a Location
1. Go to Settings → Locations
2. Click **Add Location**
3. Enter location details:
   - Name
   - Address
   - Capacity
   - Notes

### Using Locations
When creating an event, select from your saved locations for quick setup.

## Event Calendar

View all events in calendar format:
- **Month view**: See all events at a glance
- **Week view**: Detailed weekly schedule
- **Day view**: Hour-by-hour breakdown

## Event Status

Track event progress:
- **Planning**: Initial planning phase
- **Confirmed**: Event is confirmed
- **In Progress**: Event is happening now
- **Completed**: Event has finished
- **Cancelled**: Event was cancelled

## Offline Mode

Events work offline:
- ✅ View all events
- ✅ Create new events (synced when online)
- ✅ Edit events (synced when online)
- ✅ View attendees and organizations

## Tips

- **Plan ahead**: Create events early to coordinate with contacts
- **Use locations**: Save frequently-used venues for quick event creation
- **Link contacts**: Track who's attending for better networking
- **Set reminders**: Use start dates to get notifications before events
- **Add details**: Include agenda, booth numbers, or meeting rooms in descriptions
