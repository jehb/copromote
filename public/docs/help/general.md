# Promoty Help

## Welcome to Promoty

Promoty is your all-in-one promotional campaign management platform. Manage contacts, events, social media, and projects—all with offline-first capabilities.

## Key Features

### 📱 Progressive Web App (PWA)
- Install on any device (desktop, mobile, tablet)
- Works offline with automatic sync
- Fast, native-like experience

### 🔄 Offline-First
- Full functionality without internet
- Automatic background sync
- Queue system for offline changes
- Never lose your work

### 🤖 AI-Powered
- Content suggestions for social media
- Smart recommendations
- Powered by Google Gemini AI

### 📊 Comprehensive Management
- **Contacts**: Manage your professional network
- **Events**: Track promotional events and logistics
- **Social Media**: Plan and schedule content
- **Projects**: Organize campaigns and tasks

## Getting Started

### 1. Install the App

**On Desktop:**
1. Visit Promoty in Chrome, Edge, or Safari
2. Click the install icon in the address bar
3. Click "Install"

**On Mobile:**
1. Visit Promoty in your browser
2. Tap the share button
3. Select "Add to Home Screen"

### 2. Set Up AI (Optional)

1. Go to **Settings** → **AI Configuration**
2. Enter your Gemini API key
3. Test the connection
4. Start using AI features

### 3. Add Your Data

1. Start with **Contacts** - add your network
2. Create **Events** for upcoming activities
3. Plan **Social Media** content
4. Organize **Projects** and campaigns

## Offline Mode

### How It Works

Promoty stores data locally on your device:
- All data is cached for offline access
- Changes are queued when offline
- Automatic sync when reconnected
- No data loss

### What Works Offline

✅ View all data (contacts, events, posts, projects)
✅ Create new items
✅ Edit existing items
✅ Delete items
✅ Search and filter
⚠️ AI features require internet
⚠️ File uploads require internet

### Sync Status

Check the sidebar for connection status:
- **Online**: Green indicator, all features available
- **Offline**: Red indicator with pending changes count
- **Syncing**: Yellow indicator, changes uploading

## Troubleshooting

### App Not Loading
- Check internet connection
- Clear browser cache
- Try a different browser
- Refresh the page

### Sync Issues
- Verify you're online
- Check connection indicator
- Refresh the page
- Changes will retry automatically

### AI Not Working
- Verify API key in Settings
- Test connection
- Check internet connection
- Review API key permissions
