# Task Management

Stay organized with the Tasks section. Track to-dos, assignments, and deadlines across all your projects.

## Task Board
View your tasks in a Kanban board format or a list view.
- **To Do**: Tasks that haven't started yet.
- **In Progress**: Tasks currently being worked on.
- **Blocked**: Tasks pending input or other dependencies.
- **Done**: Completed tasks.

## Creating a Task
Click **New Task** to create a work item.
- Assign it to a project.
- Set a due date.
- Assign it to a team member.
- Add a detailed description.

## Filtering
Use the filters to see only tasks assigned to you, or tasks for a specific project.
