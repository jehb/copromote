# Projects Help

## Overview
Manage your promotional campaigns and initiatives. Track tasks, assets, timelines, and progress for each project.

## Creating a Project

1. Click **New Project** in the top-right corner
2. Fill in project details:
   - **Name**: Project title (required)
   - **Description**: Project goals and details
   - **Start Date**: When the project begins (required)
   - **End Date**: Target completion date
   - **Status**: Active, On Hold, or Completed

3. Click **Create** to save the project

## Managing Tasks

### Adding Tasks
1. Open a project
2. Click **Add Task**
3. Enter task details:
   - Title
   - Description
   - Due date
   - Assignee
   - Priority

### Task Status
Track task progress:
- **To Do**: Not started
- **In Progress**: Currently working on it
- **Done**: Completed

### Organizing Tasks
- Drag and drop to reorder
- Filter by status or assignee
- Sort by due date or priority

## Project Assets

### Adding Assets
1. Open a project
2. Go to **Assets** tab
3. Click **Upload Asset**
4. Select files (images, documents, videos)

### Asset Types
- **Images**: Logos, graphics, photos
- **Documents**: Briefs, contracts, presentations
- **Videos**: Promotional videos, recordings
- **Other**: Any supporting files

## Progress Tracking

### Campaign Progress Bar
- Automatically calculated based on completed tasks
- Shows percentage complete
- Updates in real-time

### Project Metrics
- **Active Tasks**: Number of tasks in progress
- **Media Assets**: Total uploaded files
- **Timeline**: Start to end date range

## Project Status

- **Active**: Currently working on this project
- **On Hold**: Temporarily paused
- **Completed**: Project finished

## Offline Mode

Projects work offline:
- ✅ View all projects
- ✅ View tasks and assets
- ✅ Create projects (synced when online)
- ✅ Update tasks (synced when online)
- ⚠️ Asset uploads require internet connection

## Tips

- **Break it down**: Create detailed tasks for better tracking
- **Set realistic dates**: Use start/end dates to plan timeline
- **Upload early**: Add assets as soon as available
- **Update status**: Keep task status current for accurate progress
- **Use descriptions**: Add context and requirements to tasks
- **Track metrics**: Monitor active tasks and completion percentage
